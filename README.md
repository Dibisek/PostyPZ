# PostyPZ

## O Projekcie

## PostyPZ to projekt zespołowy stworzony przez studentów IV roku Informatyki Politechniki Krakowskiej. Głównym celem aplikacji jest umożliwienie użytkownikom tworzenia i zarządzania treścią w nowoczesny i intuicyjny sposób. Projekt ma być funkcjonalną stroną internetową umożliwiającą:

### - Tworzenie postów.

### - Komentowanie.

### - Udostępnianie postów.

### - Lajkowanie i ocenianie treści.

# Aplikacja skupia się na prostocie użycia, przejrzystym interfejsie oraz możliwości współdzielenia treści między użytkownikami.

## Funkcjonalności

### - Tworzenie postów: Użytkownicy mogą tworzyć posty tekstowe oraz multimedialne (np. ze zdjęciami i filmami).

### - Komentarze: Każdy post może być komentowany, pozwalając na wymianę opinii i budowanie dyskusji.

### - Udostępnianie: Treści mogą być udostępniane zarówno na innych platformach, jak i w obrębie aplikacji.

### - Lajkowanie: Użytkownicy mogą wyrażać swoje uznanie poprzez kliknięcie „Lubię to”.

### - Personalizacja profilu: Każdy użytkownik posiada konto, które może dostosować do swoich preferencji.

# Technologie

## Projekt bazuje na nowoczesnych technologiach internetowych:

### Frontend: HTML5, CSS3, JavaScript

### Backend: PHP

### Baza danych: PostreSQL

### Hosting: Docker

### System kontroli wersji: Git/GitHub
